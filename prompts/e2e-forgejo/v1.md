# E2E Forgejo → Helix Pipeline Integration Test

## Goal
End-to-end integration test exercising the full Helix pipeline:
1. Agent identity provisioning in Forgejo
2. Spec decomposition via dispatcher
3. Branch + PR creation via ForgejoLoop
4. PR status verification and merge gate evaluation
5. Resource cleanup

## Test Scenarios
- Happy path: full pipeline from provisioning to PR verification
- Idempotent re-run: 409 conflict handling on branch/PR re-creation
- Error path: invalid spec decomposition failure

## Requirements
- Skippable via HELIX_E2E env var
- Forgejo reachability check before running
- t.Cleanup() for all created resources
- DryRun and live mode coverage
