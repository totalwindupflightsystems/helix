Phase 1-2: Ideation & Specification

COLLABORATION human ↔ agent
1.1 — Idea Capture

👤 Human — Change Management

  • Captures idea as text, voice, or sketch in natural language
  • Wants to know: is this aligned with roadmap? What's the rough cost?
  • Needs: links to related specs, prior art in codebase, similar rejected ideas

🤖 Agent — Verification

  • Can surface ideas from pattern detection: incidents, codebase drift, marketplace trends
  • Provides: evidence bundle linking idea to concrete data (not gut feeling)
  • Validates: does this conflict with existing ADRs? What's the blast radius?

Helix provides:

Shared Idea entity with attribution, evidence links, promotion path to Specification. Agents and humans submit ideas through the same interface. Adversarial validation via Chimera challenges every idea before it reaches the roadmap. Cost estimator runs against every candidate. Neither human nor agent can bypass the validation gate.

COLLABORATION agent ↔ agent
1.2 — Concept Validation

👤 Human — Change Management

  • Sees adversarial assessment: "3 agents flagged architectural risk, 1 identified a missing edge case from incident DB"
  • Makes the final call — agents inform, human decides
  • Sees: risk score, cost estimate, architectural fit, precedents

🤖 Agent — Verification

  • @assumption-buster enumerates and challenges every implicit assumption
  • @redteam maps attack surface from incident database history
  • @cost-auditor estimates resource budget against agent tier caps
  • Adversarial findings must cite evidence (line refs, incident IDs, benchmark data)

Helix provides:

Chimera formation for concept validation. Multi-model adversary panel with evidence-bundle output. Human sees structured risk/reward dashboard. Agent sees verifiable claims to test. Validation is a gate — concept cannot advance without passing adversarial review.

👤 Human — Change Management

  • Override justification is recorded — feeds future agent recommendations

🤖 Agent — Verification

  • Agents advocate for ideas with risk/reward evidence and historical success patterns
  • Cost estimator tags each idea with budget impact across agent tiers
  • Marketplace data enriches prioritization: "agents tagged 'database' have 92% success rate on migrations"

Helix provides:

Structured prioritization queue visible to both humans and agents. Agents rank by evidence; humans rank by strategy. Disagreement is surfaced — human overrides are documented but not blocked. Over time, prediction accuracy of both human and agent rankings feeds trust scores.
